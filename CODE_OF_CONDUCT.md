# Code of Conduct

## Our standards

Gong should be a welcoming project for people of all backgrounds and
experience levels.

Examples of constructive participation include:

- Giving specific, technical, and respectful feedback.
- Assuming good intent while discussing disagreements.
- Accepting corrections and taking responsibility for mistakes.
- Keeping discussions focused on improving the project.

Harassment, discrimination, personal attacks, threats, sexualized content,
doxxing, and sustained disruption are not acceptable.

## Enforcement

Project maintainers may edit, remove, or reject comments, commits, issues,
pull requests, and other contributions that violate these standards. Repeated
or serious violations may result in a temporary or permanent ban from project
spaces.

Report conduct concerns privately to the repository owner through the contact
methods listed on the [jenska GitHub profile](https://github.com/jenska).
Reports will be reviewed promptly and handled as confidentially as practical.

## Scope

This code applies in repository spaces and when someone is publicly
representing the project.
