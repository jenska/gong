# Security Policy

## Supported versions

Security fixes are applied to the latest version on the `master` branch.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability.

Use GitHub's private vulnerability reporting or security advisory feature:

<https://github.com/jenska/gong/security/advisories/new>

Include a description and impact, reproduction steps, affected platforms, and
any suggested mitigation. You should receive an initial response within seven
days. Please allow time for a fix and release before public disclosure.
